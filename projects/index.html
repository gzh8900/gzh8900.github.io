项目展示

一些我参与和主导的硬件项目

🔌

智能环境监测终端

基于 ESP32 + 多传感器阵列的低功耗环境监测设备，支持温湿度、PM2.5、CO2 等指标采集，通过 MQTT 上传至云端仪表盘。

ESP32MQTTFreeRTOSKiCad
📡

LoRa 无线通信模组

自研 LoRa 通信模组，支持星型组网与中继转发，空旷环境通信距离可达 3km，已应用于工业设备远程监控项目。

LoRaSX1278STM32Altium Designer
🚁

四轴飞行器飞控

从零搭建的四旋翼飞行控制系统，包含姿态解算（Madgwick AHRS）、PID 控制、无线数传等模块，基于 STM32F4 平台。

STM32F4PIDMPU6050C
🔌

智能家居中控面板

基于 ESP32-S3 + LVGL 的触控中控屏，支持 Zigbee/WiFi 双模通信，可对接 Home Assistant，自定义 UI 界面。

ESP32-S3LVGLZigbeeHome Assistant
🔧

USB 多功能调试器

集成 USB-UART、SPI 抓包、I2C 扫描、逻辑分析等功能的桌面调试工具，配套上位机软件（Python + PyQt）。

CH347PythonPyQtPCB
📟

开源示波器固件

基于国产 MCU 的迷你示波器开源固件，支持双通道采集、FFT 频谱分析、波形触发等功能，配套 3D 打印外壳。

GD32ADCFFT开源
