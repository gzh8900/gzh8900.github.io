开源示波器固件

基于国产 MCU 的迷你示波器开源固件，支持双通道采集、FFT 频谱分析、波形触发等功能，配套 3D 打印外壳。

主控GD32F303 (Cortex-M4)
采样率20MSa/s (单通道)
带宽DC ~ 5MHz
显示3.2" TFT 320×240

项目概述

一款真正开源的迷你示波器，全套资料（固件、原理图、PCB、外壳 STL、上位机）在 GitHub 公开。采用国产 GD32F303 作为主控芯片，ADC 配合外部高速比较器实现 20MSa/s 采样率。

包豪斯风格的 3D 打印外壳只需 15 克耗材，整机成本控制在 80 元以内。适合电子爱好者学习示波器原理和嵌入式开发。

固件架构

前端调理 → GD32 ADC 双通道 → DMA 乒乓缓存 → LCD 波形绘制

技术特性

📊 双通道，20MSa/s 采样
📈 FFT 频谱分析功能
🎯 边沿/脉宽/视频触发
📐 光标测量：电压/时间
💾 波形截图保存到 SD 卡
🔌 USB 通信，上位机控制

核心指标

20MSa/s最高采样率
5MHz模拟带宽
<80元整机成本
10mV/div最低灵敏度

关键代码：ADC 乒乓 DMA

#define ADC_BUF_SIZE 2048
static uint16_t adc_buf[2][ADC_BUF_SIZE];
static volatile uint8_t active_buf = 0;

void adc_dma_init(void) {
    dma_config_t dma_cfg = {
        .periph_addr = (uint32_t)&ADC_DATA_REG,
        .periph_inc = DMA_PERIPH_INC_DISABLE,
        .memory_inc = DMA_MEMORY_INC_ENABLE,
        .direction = DMA_PERIPH_TO_MEMORY,
        .buffer_size = ADC_BUF_SIZE,
    };
    // 双缓冲模式
    dma_double_buffer_init(DMA1, DMA_CH0,
        (uint32_t)adc_buf[0], (uint32_t)adc_buf[1], &dma_cfg);
    dma_interrupt_enable(DMA1, DMA_CH0, DMA_INT_FTF);
}

void DMA_IRQHandler(void) {
    if (dma_flag_get(DMA1, DMA_CH0, DMA_INT_FTF)) {
        dma_flag_clear(DMA1, DMA_CH0, DMA_INT_FTF);
        active_buf ^= 1;  // 切换缓冲区
        // 通知绘制线程
        wave_draw_request(adc_buf[active_buf ^ 1]);
    }
}
