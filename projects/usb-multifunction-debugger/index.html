USB 多功能调试器

集成 USB-UART、SPI 抓包、I2C 扫描、逻辑分析等功能的桌面调试工具，配套上位机软件（Python + PyQt）。

主控CH347 + STM32F072
协议UART/SPI/I2C/JTAG
速率UART: 3Mbps / SPI: 30MHz
上位机Python + PyQt6

项目概述

实验室常用的调试场景需要同时使用串口工具、逻辑分析仪、SPI 抓包器等设备，桌面十分凌乱。于是设计了这款桌面调试工具，将多种协议调试功能集成到一个 USB 设备中，配套统一的上位机软件。

CH347 提供 USB 转 UART/SPI/I2C/JTAG 桥接功能，STM32F072 负责扩展逻辑分析仪和协议抓包功能。

关键代码：数据采集线程

class CaptureThread(QThread):
    data_received = pyqtSignal(bytes)

    def __init__(self, device):
        super().__init__()
        self.device = device
        self.running = False

    def run(self):
        self.running = True
        buffer = bytearray()
        while self.running:
            chunk = self.device.read(4096, timeout=50)
            if chunk:
                buffer.extend(chunk)
                # 每 1KB 发射一次信号刷新 UI
                if len(buffer) >= 1024:
                    self.data_received.emit(bytes(buffer))
                    buffer.clear()
        # 发送剩余数据
        if buffer:
            self.data_received.emit(bytes(buffer))
