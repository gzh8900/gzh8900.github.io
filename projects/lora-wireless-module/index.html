LoRa 无线通信模组

自研 LoRa 通信模组，支持星型组网与中继转发，空旷环境通信距离可达 3km，已应用于工业设备远程监控项目。

射频芯片SX1262
通信距离3km（开阔）
频段CN470 / EU868
功耗TX:45mA / Sleep:0.6μA

项目概述

针对工业设备远程监控场景，设计了一款兼容 LoRaWAN 协议的通信模组。支持自适应数据速率（ADR），内置中继转发算法以扩展网络覆盖。已在一个 500+ 节点的园区项目中部署运行超 12 个月。

网络拓扑

终端节点 ×500 ⬆⬇ 中继网关 ×8 ⬆⬇ LoRaWAN Server → 云端应用

技术特性

📶 自适应 SF/SF 速率调整
🔄 中继转发 ≤3 跳
🔋 终端节点续航 >3 年
🛡️ AES-128 端到端加密
📊 在线 RSSI 信号热力图
🔧 空中升级（OTAA）

核心指标

3km通信距离

-137dBm灵敏度
0.6μA待机电流

固件架构设计

模组固件基于 FreeRTOS，分为三层架构：

  • 硬件抽象层 (HAL)：封装 SX1262 SPI 驱动、GPIO 控制和定时器
  • 协议栈层：LoRaWAN 1.0.4 协议实现，支持 Class A/C 模式
  • 应用层：传感器数据采集、中继路由算法、OTA 升级管理

关键代码：中继转发

typedef struct {
    uint8_t  hop_count;
    uint32_t src_addr;
    uint32_t dst_addr;
    uint8_t  payload[64];
} __attribute__((packed)) relay_frame_t;

bool relay_handle(relay_frame_t *frame) {
    if (frame->hop_count >= MAX_HOPS) return false;

    // 检查是否已转发过（去重）
    if (relay_dup_check(frame->src_addr, frame->payload)) return false;

    frame->hop_count++;
    return lora_send(frame, sizeof(relay_frame_t) + frame->payload[0]);
}
