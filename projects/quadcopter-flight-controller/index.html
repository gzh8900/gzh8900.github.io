四轴飞行器飞控

从零搭建的四旋翼飞行控制系统，包含姿态解算（Madgwick AHRS）、PID 控制、无线数传等模块，基于 STM32F4 平台。

主控STM32F405RGT6
IMUMPU6050 + HMC5883L
高度计BMP280
无线NRF24L01+ PA

项目概述

大学时期开始的自研飞控项目，历时 18 个月完成从原型到稳定飞行的全过程。实现了 Madgwick AHRS 姿态解算算法、串级 PID 控制、遥控器 PPM 信号解析、无线数传地面站等功能。飞控代码完全开源。

控制流程

遥控器 PPM → STM32 输入捕获 → 姿态解算 AHRS → PID 控制器 → PWM 输出 → 电调

技术特性

🧮 Madgwick AHRS 姿态融合，更新率 1kHz
🎛️ 串级 PID：外环角度 / 内环角速度
📡 2.4GHz NRF24L01 数传，距离 500m+
📊 地面站软件（Python + PyQt）
🔄 6 种飞行模式：自稳/定高/返航
📝 黑匣子日志记录

核心指标

1kHz姿态更新率
250HzPID 控制环
±0.5°悬停角度精度
25min悬停续航

关键代码：Madgwick AHRS

void madgwick_update(madgwick_t *f, float gx, float gy, float gz,
                     float ax, float ay, float az, float dt) {
    float q1 = f->q1, q2 = f->q2, q3 = f->q3, q4 = f->q4;

    // 归一化加速度计数据
    float norm = sqrtf(ax*ax + ay*ay + az*az);
    if (norm < 0.001f) return;
    ax /= norm; ay /= norm; az /= norm;

    // 梯度下降法补偿陀螺漂移
    float s1 = 2.0f * (q2*q4 - q1*q3) - ax;
    float s2 = 2.0f * (q1*q2 + q3*q4) - ay;
    float s3 = 2.0f * (0.5f - q2*q2 - q3*q3) - az;

    // 四元数更新
    q1 += (-q2*gx - q3*gy - q4*gz - f->beta * s1) * dt;
    q2 += ( q1*gx + q3*gz - q4*gy - f->beta * s2) * dt;
    q3 += ( q1*gy - q2*gz + q4*gx - f->beta * s3) * dt;
    q4 += ( q1*gz + q2*gy - q3*gx) * dt;

    // 归一化
    norm = sqrtf(q1*q1 + q2*q2 + q3*q3 + q4*q4);
    f->q1 = q1/norm; f->q2 = q2/norm;
    f->q3 = q3/norm; f->q4 = q4/norm;
}
