智能环境监测终端

基于 ESP32 + 多传感器阵列的低功耗环境监测设备，支持温湿度、PM2.5、CO₂ 等指标实时采集，通过 MQTT 上传至云端仪表盘。

平台ESP32-WROOM-32
连接WiFi / MQTT
传感器SHT30 + PMS5003 + MH-Z19B
状态已量产

项目概述

为智慧农业园区设计的分布式环境监测节点。每 15 分钟采集一次温湿度、PM2.5、CO₂ 和光照强度，通过 MQTT 上报至 ThingsBoard 仪表盘。整机功耗优化至平均 45μA，一节 18650 电池可续航 6 个月以上。

系统架构

传感器阵列 → ESP32 (FreeRTOS) → MQTT Broker → ThingsBoard

技术特性

🔋 深度睡眠 5μA，15分钟唤醒周期
📊 板载 OLED 实时显示数据
📡 WiFi MQTT 自动重连机制
🌡️ -20°C ~ 60°C 宽温工作
🔒 看门狗 + 异常复位保护
📱 OTA 远程固件升级

核心指标

45μA平均功耗
6个传感器通道
15min采样周期
IP65防护等级

关键代码：MQTT 自动重连

static void mqtt_app_start(void)
{
    esp_mqtt_client_config_t mqtt_cfg = {
        .uri = CONFIG_BROKER_URI,
        .keepalive = 60,
        .disable_auto_reconnect = false,
    };
    client = esp_mqtt_client_init(&mqtt_cfg);
    esp_mqtt_client_register_event(client, ESP_EVENT_ANY_ID, mqtt_event_handler, NULL);
    esp_mqtt_client_start(client);
}

static void mqtt_event_handler(void *handler_args, esp_event_base_t base,
                                int32_t event_id, void *event_data)
{
    switch (event_id) {
        case MQTT_EVENT_CONNECTED:
            ESP_LOGI(TAG, "MQTT connected");
            esp_mqtt_client_subscribe(client, "/sensor/data", 1);
            break;
        case MQTT_EVENT_DISCONNECTED:
            ESP_LOGW(TAG, "MQTT disconnected, will auto-reconnect");
            break;
        default: break;
    }
}
