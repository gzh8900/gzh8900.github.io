智能家居中控面板

基于 ESP32-S3 + LVGL 的触控中控屏，支持 Zigbee/WiFi 双模通信，可对接 Home Assistant，自定义 UI 界面。

主控ESP32-S3 (2.4GHz)
屏幕4.0" TFT 480×480
通信WiFi + Zigbee (CC2652)
协议MQTT / Home Assistant

项目概述

一款壁挂式智能家居中控面板，用于替代传统的墙壁开关。支持灯光控制、窗帘开关、温度调节、场景联动等功能。基于 ESP32-S3 驱动 LVGL 图形界面，通过 CC2652 协处理器实现 Zigbee 网络控制。

系统架构

LVGL 触控 UI ↕ ESP32-S3 主控 ↕ CC2652 Zigbee ↕ Zigbee 设备

技术特性

🖥️ LVGL v8 UI，480×480 圆形屏
🔗 双模通信：WiFi + Zigbee 3.0
🏡 Home Assistant 原生对接
🎨 用户可自定义主题和场景
🗣️ 支持语音助手联动
🌙 光线感应自动调节亮度

核心指标

4.0"触控屏幕
100+可接入设备
1.2W典型功耗
≤50ms响应延迟

关键代码：LVGL 页面切换

static lv_obj_t *ui_scr_main;
static lv_obj_t *ui_panel_light;
static lv_obj_t *ui_btn_left;

static void btn_left_click(lv_event_t *e) {
    static uint8_t page = 0;
    page = (page + 1) % 5;

    lv_obj_t *pages[] = {ui_panel_light, ui_panel_curtain,
                         ui_panel_temp, ui_panel_scene, ui_panel_setting};

    for (int i = 0; i < 5; i++) {
        lv_obj_add_flag(pages[i], LV_OBJ_FLAG_HIDDEN);
    }
    lv_obj_clear_flag(pages[page], LV_OBJ_FLAG_HIDDEN);
}

void ui_init(void) {
    ui_scr_main = lv_obj_create(NULL);
    lv_scr_load(ui_scr_main);

    ui_btn_left = lv_btn_create(ui_scr_main);
    lv_obj_set_size(ui_btn_left, 50, 50);
    lv_obj_align(ui_btn_left, LV_ALIGN_LEFT_MID, 10, 0);
    lv_obj_add_event_cb(ui_btn_left, btn_left_click, LV_EVENT_CLICKED, NULL);
}
